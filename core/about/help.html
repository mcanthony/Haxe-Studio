Haxe Studio Help

About

Haxe Studio is a text editor for Haxe Project. It tries to integrate with your project by reading haxestudio.config file in your project folder. Currently , Haxe Studio supports :

  • Lime
    • OpenFL
    • HaxeFlixel
    • HaxePunk

First Run

Haxe Studio is distributed using the update system, but you need the core file first. The core file are the node-webkit runtime and some script to help you run. Simply follow this guideline :

  1. Download the core here
  2. Extract the core to the lowest folder of your hard drive a.k.a c:\ or ~
  3. Run Haxe-Studio-Update (.bat/.sh/.command depends on your OS)
  4. Click Check for Update
  5. Click Download Update
  6. Click Download Update and wait for the patch to be downloaded
  7. Click Apply Update
  8. Close Haxe-Studio-Update
  9. Run Haxe-Studio (.bat/.sh/.command depends on your OS)
  10. Done!

How to use

Reminder : Haxe Studio only works with a project. If you want to edit a File, please use ATOM editor with Haxe plugin, or any other editors.

Working with Haxe Studio might be different from your favourite editor. So, we suggest you follow this guideline

  1. Create or Open a project (H/XML)
  2. Choose your file to edit
  3. Write your code
  4. Build/Compile
  5. repeat from #2

Integrate your Project

If you tried to open a non-compatible project (H/XML), you'll notice it didn't integrate well with Haxe Studio. To remedy this, You can simply add haxestudio.config file to your non-compatible project to make it compatible.

{
"compiler":"lime",
"base_library":"openfl",
"source_folder":"Source"
}
  • compiler :

    The guy who compile your code to the respected target. Supported compiler are written in About Section

  • base_library :

    The library you depends on. This integrate with the new file menu and show you the template.

  • source_folder :

    Your -src folder. This integrate with the new file menu and handle the file creation
